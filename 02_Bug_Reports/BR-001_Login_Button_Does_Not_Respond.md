# BR-001 Login Button Does Not Respond
## Bug ID
BR-001
## Title
Login button deos not respond after entering valid credentials.
## Environment
- Application : Demo Login Page
- Browser: Chrome (Latest version)
- OS: Windows 11

## Preconditions
- User is on the login page.
- User has valid credentials.

## Steps to Reproduce
1.Navigate to Login page.

2.Enter valid username.

3.Enter valid Password.

4.Click Login button.

## Actual Result
No action is performed after clicking Login button. User remains on the login page.
## Expected Result
User should be authenticated and redirected to Dashboard page.
