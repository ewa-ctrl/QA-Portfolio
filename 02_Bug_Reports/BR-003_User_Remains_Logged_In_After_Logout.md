# BR-004 User Remains Logged In After Logout
## Summary
User can still access protected pages after login out.
## Environment
- Browser: Chrome 137
- OS: Windows 11
## Preconditions
User is logged in.
## Steps to Reproduce 

1.Log in to the application.  

2.Click Logout.

3.Copy the URL of Dashboard.

4.Paste the URL into browser addres bar.

## Actual Result
Dashboard page is displayed.
## Expected Result 
User should be redirected to login page and denied access.
## Severity
Critical
## Priority
High
## Status
Open
