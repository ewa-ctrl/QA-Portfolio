# BR-002 Error Message Overlaps Login Form
## Summary 
Error message overlaps login form elements when invalid credentials are entered.
## Environment
- Browser: Chrome 137
- OS: Windows 11
## Preconditions
User is on Login page.
## Steps to Reproduce
1.Open Login Page.

2.Enter invalid username.

3.Enter invalid password.

4.Click Login button.

## Actual Result
Error message overlaps username and password fields.
## Expected Result
Error message should be displayed below the login form without overlaping any elements.
## Severity
Medium
## Priority
Medium
## Status
Open
