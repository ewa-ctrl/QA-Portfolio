# Bug Reports

This folder contains sample bug reports created during manual testing exercises.

## Bug Reports Included

- BR-001 Login Button Does Not Respond
- BR-002 Error Message Overlaps Login Form
- BR-003 User Remains Logged In After Logout
- BR-004 Password Reset Link Not Sent

## Information Included

Each bug report contains:

- Bug ID
- Title
- Severity
- Priority
- Environment
- Preconditions
- Steps to Reproduce
- Expected Result
- Actual Result

## Purpose

The purpose of these reports is to demonstrate bug identification, documentation, and communication skills used in software testing.
