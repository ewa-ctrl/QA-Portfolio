# Test Cases

This folder contains manual test cases created for a login functionality.

## Test Cases Included

- TC-001 Successful Login
- TC-002 Invalid Password
- TC-003 Empty Username
- TC-004 Empty Password
- TC-005 Multiple Failed Login Attempts

## Purpose

The objective of these test cases is to verify the correctness, security, and usability of the login process.

## Test Design Techniques Used

- Positive Testing
- Negative Testing
- Boundary and Validation Checks
