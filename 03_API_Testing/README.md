# API Testing

This folder contains API testing exercises performed using Postman.

## API Scenarios Included

- API-001 GET All Posts
- API-002 GET Single Post
- API-003 GET Non Existing Post
- API-004 Create Post
- API-005 Update Post
- API-006 Delete Post

## Tools

- Postman
- JSONPlaceholder API

## Covered Areas

- GET requests
- POST requests
- PUT requests
- DELETE requests
- Response validation
- Status code verification
- JSON response analysis
